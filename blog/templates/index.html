What is a client?

A client is whoever is trying to access data from a server .

What is a server?

A server is used to store and distribute data to clients through requests.

What is an HTTP request?

Is used to allow for communication between client and server. The client sends a request to the server asking for information.

What is a response?

An HTTP response is what the server sends back to the client following a request. The response contains codes related to the outcome of the request and could also contain the requested information.

What is Django and why do we use it?

Django is a python framework that provides us with the structure and methods for building web apps
